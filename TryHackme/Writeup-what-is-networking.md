## Qué aprendí
- Una red es un grupo de dispositivos conectados entre sí
- Las direcciones IP identifican dispositivos en la red
- Las direcciones MAC identifican el hardware físico
- El protocolo ping usa ICMP para comprobar conectividad

## Herramientas usadas
- Wireshark para capturar tráfico real
- Terminal para usar el comando ping

## Flags obtenidas
- THM{I_PINGED_THE_SERVER}
