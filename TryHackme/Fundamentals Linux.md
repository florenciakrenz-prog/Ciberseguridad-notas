# Linux Fundamentals Part 1

## Resumen

Esta sala cubre los fundamentos básicos de Linux: navegación por el 
sistema de archivos, comandos esenciales de terminal, y conceptos básicos 
necesarios para trabajar cómodamente en un entorno Linux.

## Conceptos clave

- **Navegación del sistema de archivos:** comandos como `cd`, `ls`, `pwd` 
  para moverse y orientarse dentro de la estructura de directorios.
- **Comandos básicos de terminal:** manipulación de archivos y carpetas, 
  visualización de contenido.
- **Estructura de directorios de Linux:** entender la jerarquía estándar 
  (`/home`, `/var`, `/etc`, etc.) que ya se había visto parcialmente en 
  "Registro de Linux para SOC".

## Conexión con lo ya aprendido

Esta sala formaliza y refuerza los comandos que ya venía usando de forma 
práctica en "Registro de Linux para SOC" (como `cat`, `grep`, `ls -la`), 
dándoles una base teórica más sólida.

## Qué aprendí

Una base más firme de comandos y navegación en Linux, que hace mucho más 
fluido el trabajo de investigación de logs y sistemas que se viene 
practicando en las salas anteriores.
