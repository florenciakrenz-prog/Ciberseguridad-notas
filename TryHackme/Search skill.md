# Search Skills

## Resumen

Esta sala cubre técnicas de búsqueda efectivas para investigaciones de 
ciberseguridad: cómo formular búsquedas precisas, usar operadores de 
búsqueda avanzada, y pensar de forma crítica al investigar conceptos, 
herramientas o indicadores de compromiso (IOCs) desconocidos.

## Conceptos clave

- **Búsqueda efectiva:** formular queries específicas en vez de preguntas 
  genéricas, usando palabras clave técnicas precisas.
- **Pensamiento crítico aplicado a la investigación:** evaluar fuentes, 
  cruzar información de múltiples resultados, y no confiar ciegamente en 
  el primer resultado.

## Qué aprendí

Esta es una habilidad transversal que se usa constantemente como analista 
SOC — al investigar un hash sospechoso, un dominio desconocido, un proceso 
inusual, o una herramienta como `naabu` (vista en la sala anterior), saber 
buscar de forma rápida y precisa ahorra tiempo y mejora la calidad de la 
investigación.

## Conexión con lo ya aprendido

Esta habilidad complementa directamente el trabajo de investigación 
forense practicado en "Registro de Linux para SOC" — cuando aparece algo 
desconocido en un log (como `naabu` o una IP sospechosa), saber buscarlo 
eficientemente es el siguiente paso lógico después de identificarlo.
